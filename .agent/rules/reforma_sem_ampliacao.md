# Regra: Reforma Sem Ampliação

## Descrição
Todos os projetos de reforma gerados por este sistema devem considerar **apenas reformas sem ampliação de área construída**.

## Restrições

### ❌ NÃO PERMITIDO
- Ampliação de área construída
- Construção de novos cômodos
- Fechamento de varandas (exceto cortina de vidro removível)
- Alteração de fachada que aumente área
- Construção de mezaninos
- Anexos ou edículas novas

### ✅ PERMITIDO
- Demolição de paredes internas (não estruturais, com ART)
- Integração de ambientes existentes
- Reforma de acabamentos
- Atualização de instalações elétricas e hidráulicas
- Troca de revestimentos, pisos e forros
- Instalação de móveis planejados
- Automação residencial
- Impermeabilização
- Manutenção de telhados

## Justificativa
- Reformas sem ampliação geralmente não requerem aprovação de projeto na prefeitura
- Menor complexidade de licenciamento
- Escopo bem definido para orçamentação
- Alinhamento com o público-alvo (proprietários de imóveis existentes)

## Aplicação
Esta regra deve ser aplicada em:
1. Geração de escopos de reforma
2. Sugestões de projetos
3. Quantificação de serviços
4. Exportação de JSONs

## Exceções
Nenhuma exceção permitida sem aprovação explícita do usuário.
