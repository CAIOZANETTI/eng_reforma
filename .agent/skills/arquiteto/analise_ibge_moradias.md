# Análise IBGE - Panorama das Moradias no Brasil

> **Documento de Referência para Reformas**  
> *Dados oficiais do Censo Demográfico 2022 e PNAD Contínua 2023*  
> *Fonte: Instituto Brasileiro de Geografia e Estatística (IBGE)*

---

## Resumo Executivo

O Censo Demográfico 2022 revelou um Brasil com **72,5 milhões de domicílios ocupados** e **90,7 milhões de domicílios totais** (incluindo vagos e de uso ocasional). Este documento compila os principais indicadores habitacionais relevantes para o mercado de reformas, oferecendo dados para dimensionamento de demanda, tipologias de projeto e perfil econômico dos clientes.

### Destaques para o Mercado de Reformas

| Indicador | Valor | Implicação para Reformas |
|-----------|-------|-------------------------|
| Domicílios totais | 90,7 milhões | Mercado potencial gigantesco |
| Casas | 84,8% da população | Foco principal em reformas residenciais unifamiliares |
| Apartamentos | 12,5% da população (crescendo) | Segmento em expansão, NBR 16.280 obrigatória |
| Média de moradores/domicílio | 2,79 pessoas | Famílias menores = imóveis sendo adaptados |
| Domicílios sem rede de esgoto | 24,3% (~49 milhões pessoas) | Demanda por reformas hidráulicas/sanitárias |
| Paredes de alvenaria com revestimento | 87% | Base de reformas é estrutura consolidada |

## 11. Checklist para Uso nas Reformas

### Ao Avaliar um Projeto de Reforma:

- [ ] Verificar tipo de imóvel (casa/apto) → define NBR 16.280
- [ ] Identificar região → ajusta expectativa de materiais e mão de obra
- [ ] Consultar material das paredes → define técnicas de abertura/fechamento
- [ ] Verificar situação de esgoto → pode requerer adequação
- [ ] Estimar classe social do cliente → define acabamento e orçamento
- [ ] Considerar número de cômodos → dimensiona escopo

### Perguntas-Chave por Perfil:

| Perfil | Perguntas | Fonte de Dados |
|--------|-----------|----------------|
| **Apartamento SP** | NBR 16.280? Convenção? | Censo + Legislação |
| **Casa Nordeste** | Material das paredes? Rede de esgoto? | Censo Regional |
| **Classe C** | Orçamento limitado? Prioridades? | PNAD + Pesquisas |
| **Imóvel antigo** | Instalações atualizadas? Fissuras estruturais? | Avaliação in loco |
